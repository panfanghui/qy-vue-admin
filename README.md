#### 介绍
前端框架使用的是veu +element UI 
#### 软件架构
软件架构说明


#### 安装教程

1.  安装淘宝镜像
    npm install -g cnpm --registry=https://registry.npm.taobao.org
2.  cnpm i
3.  开发模式运行
    npm run serve
    打包
    npm run build
4. 修改接口域名地址在src/config/settings.js中设置即可    

#### 使用说明

1.  开发模式运行
    npm run serve
2.  快速删除node_module
    安装 npm install rimraf -g
    删除 rimraf node_modules
